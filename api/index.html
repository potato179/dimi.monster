디미고 API 저장소

1. 디미고인 토큰
URL: https://api.dimi.monster/auth
요청 형식: POST(text)
요청 요소: username, password
응답 형식: JSON

2. 디미고인 사용자 정보 (토큰 형식)
URL: https://api.dimi.monster/me
요청 형식: POST(text)
요청 요소: accessToken
응답 형식: JSON

3. 디미고인 사용자 정보 (아이디 형식)
URL: https://api.dimi.monster/authMe
요청 형식: POST(text)
요청 요소: username, password
응답 형식: JSON

4. 디미고 급식
URL: https://api.dimi.monster/meal
요청 형식: GET
요청 요소: date (yyyy-MM-dd 형식)
응답 형식: JSON

5. 디미고 시간표
URL: https://api.dimi.monster/timetable
요청 형식: GET
요청 요소: grade, class
응답 형식: JSON
